# Task 1: Iris Dataset Analysis and Visualization

## Objective

Explore and visualize the Iris dataset to understand relationships between features and gain insights through data analysis.

## Approach

* Loaded the Iris dataset using Pandas.
* Examined dataset structure using shape, columns, and head functions.
* Performed Exploratory Data Analysis (EDA).
* Created scatter plots, histograms, and box plots using Matplotlib and Seaborn.
* Analyzed feature distributions and relationships among variables.

## Results and Insights

* Different Iris species show distinct patterns in petal and sepal measurements.
* Scatter plots revealed strong relationships between petal dimensions.
* Histograms helped understand feature distributions.
* Box plots highlighted data spread and potential outliers.
* The project provided a solid foundation in data exploration and visualization.
